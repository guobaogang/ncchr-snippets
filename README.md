# ncchr-snippets README

用友NCCHR项目开发提示工具

## Features

## editTable方法
# create-editTable 创建编辑表格
# editTable-delete-row 编辑表格删除行
# editTable-get-all-rows 获取编辑表格内所有行数据
# eidtTable-get-all-data 获取编辑表格内全数据
# editTable-add-row 通过index值新增行
# editTable-hide-col-by-key 根据id和column的key隐藏某列
# editTable-show-col-by-key 根据id和column的key显示某列
# editTable-set-value-by-key-index 根据行序号设置表格某行某个字段值
# editTable-set-table-data 设置编辑表格数据
# editTable-cancel-edit 取消编辑，还原数据
# editTable-set-status 设置表格编辑态/浏览态
# editTable-get-changed-rows 获取表格修改过的数据
# editTable-set-value-by-key-rowid 根据rowid设置表格某行某个字段值
# editTable-get-value-by-key-index 根据index获取表格某行某个字段值
# editTable-get-value-by-key-rowid 根据rowId设置获取某行某个字段值
# editTable-set-editable-by-key 通过rowid和键设置表格某行某个字段编辑性
# editTable-get-column-value 获取表格某列数据
# editTable-set-column-value 设置表格某列数据
# editTable-get-checked-value 获取表格选中行
# editTable-select-all-rows 选中/取消选中表格所有行
# editTable-delete-rows-by-index 通过index删除行
# editTable-delete-rows-by-rowid 通过rowid删除行
# editTable-focus-row-by-index focus某一行
# editTable-set-row-by-index 通过index设置一行数据
# editTable-set-editable-row-by-rowid 通过rowId设置表格某些行的编辑性
# editTable-set-editable-row-by-index 通过Index设置表格某些行的编辑性
# editTable-set-editable-rowkey-by-rowid 通过rowid设置表格某些行的某些字段编辑性
# editTable-set-editable-rowkey-by-index 通过index设置表格某些行的某些字段编辑性
# editTable-set-checkbox-disabled 设置每行第一个Checkbox是否禁用
# editTable-set-all-checkbox-able 设置设置所有多选框禁用和启用的方法
# editTable-select-table-rows 设置某些行的选中状态
# editTable-check-required 检验必输项的方法
# editTable-set-editable-key-edit 统一设置表格字段的编辑性

## form方法
# create-form 创建表单
# form-get-all-value 获取表单所有数据
# form-set-data-value 设置表单全部数据
# form-cancel 表单取消编辑
# form-set-status 设置表单status
# form-get-status 获取编辑状态
# form-empty-all-value 清空表单所有数据
# form-check-required 检验是否必填
# form-get-form-items-value 获取表单中某个字段的值
# form-set-form-items-value 设置表单中某个字段的值
# form-set-form-items-disabled 设置表单编辑性
# form-set-form-items-visible 设置表单可见性
# form-set-form-items-required 设置表单某些字段的必输性
